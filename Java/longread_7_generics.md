## 1. –í–≤–µ–¥–µ–Ω–∏–µ –∏ —Ü–µ–ª–∏

**Generics** –≤ Java ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–± —É–±—Ä–∞—Ç—å –¥—É–±–ª–∏—Ä–æ–≤–∞–Ω–∏–µ –∫–æ–¥–∞ –±–µ–∑ –ø–æ—Ç–µ—Ä–∏ **type safety**, —Å–¥–µ–ª–∞—Ç—å API –≤—ã—Ä–∞–∑–∏—Ç–µ–ª—å–Ω–µ–µ –∏ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—Ç–∏—Ç—å –æ—à–∏–±–∫–∏ –≤—Ä–µ–º–µ–Ω–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ª–∏—à–Ω–∏–µ –∫–∞—Å—Ç—ã –∏ –Ω–µ–æ–∂–∏–¥–∞–Ω–Ω—ã–µ `ClassCastException`).

#### –ß—Ç–æ —Ä–∞–∑–±–µ—Ä—ë–º

- **–ù–∞—Å–ª–µ–¥–æ–≤–∞–Ω–∏–µ –∏ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è (bounds)**: `<T extends ‚Ä¶>`, –º–Ω–æ–∂–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ –≥—Ä–∞–Ω–∏—Ü—ã `&`, –≤–ª–∏—è–Ω–∏–µ –Ω–∞ —Å–∏–≥–Ω–∞—Ç—É—Ä—ã.
- **–ü–æ–¥—Å—Ç–∞–Ω–æ–≤–æ—á–Ω—ã–µ —Ç–∏–ø—ã (wildcards) –∏ –ø—Ä–∞–≤–∏–ª–æ PECS**: `?`, `? extends T` (producer), `? super T` (consumer), –∫–æ–≥–¥–∞ wildcards –Ω–µ –ø–æ–¥—Ö–æ–¥—è—Ç.

- **Heap pollution –∏ varargs**: –æ—Ç–∫—É–¥–∞ –±–µ—Ä—ë—Ç—Å—è –ø—Ä–µ–¥—É–ø—Ä–µ–∂–¥–µ–Ω–∏–µ, –∫–æ–≥–¥–∞ —É–º–µ—Å—Ç–Ω–∞ `@SafeVarargs`, –±–µ–∑–æ–ø–∞—Å–Ω—ã–µ —à–∞–±–ª–æ–Ω—ã.
- **–î–æ generics: raw types (—Å—ã—Ä—ã–µ —Ç–∏–ø—ã)**: –ø–æ—á–µ–º—É `List<?>` –ø—Ä–µ–¥–ø–æ—á—Ç–∏—Ç–µ–ª—å–Ω–µ–µ `List`.

## 3. Generic-—Ç–∏–ø—ã –∏ —Å–æ–∑–¥–∞–Ω–∏–µ —ç–∫–∑–µ–º–ø–ª—è—Ä–æ–≤
