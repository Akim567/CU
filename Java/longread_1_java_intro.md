## –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏

#### –û—Å–Ω–æ–≤–Ω—ã–µ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã:

- **OR (`||`)** ‚Äî –∏—Å—Ç–∏–Ω–∞, –µ—Å–ª–∏ —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ. *Short-circuit*: –µ—Å–ª–∏ —Å–ª–µ–≤–∞ `true`, –ø—Ä–∞–≤–∞—è —á–∞—Å—Ç—å –Ω–µ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è.
- **NOT (`!`)** ‚Äî –∏–Ω–≤–µ—Ä—Å–∏—è –∑–Ω–∞—á–µ–Ω–∏—è `boolean`.

```java

System.out.println(a || b); // true
System.out.println(!a);     // false
```

*   `a && b` ‚Üí `true && false` ‚Üí `false`.

*   `a || b` ‚Üí `true || false` ‚Üí `true`.

*   `!a` ‚Üí `!true` ‚Üí `false`.

## –ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤

–í Java —Ç–∏–ø—ã —Å—Ç—Ä–æ–≥–æ —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω—ã, –∏ –∏–Ω–æ–≥–¥–∞ —Ç—Ä–µ–±—É–µ—Ç—Å—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –æ–¥–Ω–æ–≥–æ –ø—Ä–∏–º–∏—Ç–∏–≤–Ω–æ–≥–æ —Ç–∏–ø–∞ –≤ –¥—Ä—É–≥–æ–π. –≠—Ç–æ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è type casting (–∫–∞—Å—Ç–∏–Ω–≥).

1. **–ù–µ—è–≤–Ω–æ–µ, —Ä–∞—Å—à–∏—Ä—è—é—â–µ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ (widening)**

–¢–∞–∫–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏, –±–µ–∑ —è–≤–Ω–æ–≥–æ —É–∫–∞–∑–∞–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–∞. –û–Ω–æ –≤—Å–µ–≥–¥–∞ –±–µ–∑–æ–ø–∞—Å–Ω–æ, –ø–æ—Ç–æ–º—É —á—Ç–æ –º–µ–Ω—å—à–∏–π –ø–æ –¥–∏–∞–ø–∞–∑–æ–Ω—É —Ç–∏–ø –ø—Ä–∏–≤–æ–¥–∏—Ç—Å—è –∫ –±–æ–ª—å—à–µ–º—É, –±–µ–∑ –ø–æ—Ç–µ—Ä–∏ –¥–∞–Ω–Ω—ã—Ö.


* int (32 –±–∏—Ç–∞) ‚Üí long (64 –±–∏—Ç–∞).

2. **–Ø–≤–Ω–æ–µ, —Å—É–∂–∞—é—â–µ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ (narrowing)**

* –ó–¥–µ—Å—å long (64 –±–∏—Ç–∞) –ø—Ä–∏–≤–æ–¥–∏—Ç—Å—è –∫ int (32 –±–∏—Ç–∞).
* –ï—Å–ª–∏ l –±–æ–ª—å—à–µ –¥–∏–∞–ø–∞–∑–æ–Ω–∞ int, –ø—Ä–æ–∏–∑–æ–π–¥—ë—Ç –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏–µ, —Ä–µ–∑—É–ª—å—Ç–∞—Ç –±—É–¥–µ—Ç –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã–º (wrap-around).

* boolean –Ω–µ —É—á–∞—Å—Ç–≤—É–µ—Ç –≤ –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è—Ö ‚Äî –µ–≥–æ –Ω–µ–ª—å–∑—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –Ω–∏ –≤ —á–∏—Å–ª–∞, –Ω–∏ –∏–∑ —á–∏—Å–µ–ª.
* –£ –æ–ø–µ—Ä–∞—Ü–∏–π –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è –µ—Å—Ç—å —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –ø—Ä–∞–≤–∏–ª–∞ –≤ Java Language Specification (JLS), –∫–æ—Ç–æ—Ä—ã–µ –æ–ø—Ä–µ–¥–µ–ª—è—é—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –¥–∞–∂–µ –ø—Ä–∏ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—è—Ö (–≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä, C/C++, –≥–¥–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –º–æ–∂–µ—Ç –∑–∞–≤–∏—Å–µ—Ç—å –æ—Ç –ø–ª–∞—Ç—Ñ–æ—Ä–º—ã).
* –° char –≤–∞–∂–Ω–æ –ø–æ–º–Ω–∏—Ç—å: —ç—Ç–æ 16-–±–∏—Ç–Ω—ã–π –±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–π —Ç–∏–ø. –ï–≥–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤ int –¥–∞—ë—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ, –∞ –Ω–µ —Å–∏–º–≤–æ–ª.

- –£—Å–ª–æ–≤–∏–µ –¥–æ–ª–∂–Ω–æ –∏–º–µ—Ç—å —Ç–∏–ø **boolean** (–Ω–∏–∫–∞–∫–∏—Ö ¬´truthy/falsey¬ª –∑–Ω–∞—á–µ–Ω–∏–π, –∫–∞–∫ –≤ Python –∏–ª–∏ JS).

–ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –≤–µ—Ç–≤–ª–µ–Ω–∏—è –ø–æ –∑–Ω–∞—á–µ–Ω–∏—é:

**–û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:**
- –í —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã—Ö –≤–µ—Ä—Å–∏—è—Ö Java (–Ω–∞—á–∏–Ω–∞—è —Å 12, —Ñ–∏–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω–æ –≤ 14) `switch` –º–æ–∂–µ—Ç –±—ã—Ç—å –∫–∞–∫ **statement** (–ø—Ä–æ—Å—Ç–æ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –∫–æ–¥), —Ç–∞–∫ –∏ **expression** (–≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ).
